This page is the library entrance to Karyon’s technical corpus. Instead of one continuous documentation tree, the knowledge base is organized as separate books that share one search index while keeping each reading experience scoped to its own table of contents.

If you are trying to understand what is implemented today rather than what the organism is ultimately meant to become, pair this section with the [Project Status](/status/) page.

## Available books

Use this page the way you would use the notes index: scan the available books, choose the one that matches your question, and then move into that book’s own reading flow.
