The architecture book is one of the strongest parts of the project, but it does double duty today: it explains the intended organism and also acts as a guide for the codebase that exists now.

The new public scaffold tries to separate those jobs more clearly. The docs remain central, but they now live beside explicit status pages so readers can keep theory and implementation in view at the same time.
